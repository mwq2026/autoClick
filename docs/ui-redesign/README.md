## 技术实现

### 修改的文件

1. **src/main.cpp**
   - 替换 `ImGui::StyleColorsDark()` 为自定义浅色主题
   - 配置完整的 ImGui 颜色方案
   - 优化样式参数（内边距、间距等）
   - 修改背景清除颜色

2. **src/app/App.cpp**
   - 重构 `DrawSimpleMode()` - 卡片式布局
   - 重构 `DrawAdvancedMode()` - 卡片式布局
   - 优化 `OnFrame()` - 头部和模式切换
   - 美化 `DrawStatusBar()` - 带背景的状态栏
   - 更新 `DrawAppLogo()` - 浅色主题适配

### 代码特点

- 保持原有功能完全不变
- 使用 ImGui 原生功能，无外部依赖
- 代码结构清晰，易于维护
- 通过 `ImGui::BeginChild` 实现卡片布局
- 使用 `ImDrawList` 绘制自定义背景

## 简单模式卡片

1. **文件配置卡片** (120px)
   - 文件路径输入
   - 浏览/另存为按钮

2. **回放设置卡片** (90px)
   - 屏蔽输入选项
   - 倍速滑块

3. **控制按钮卡片** (80px)
   - 录制/加载/回放按钮（居中）

4. **统计信息卡片** (自适应)
   - 事件统计
   - 回放进度

## 高级模式卡片

1. **Lua 文件配置卡片** (100px)
   - 文件路径和浏览
   - 打开/保存按钮

2. **脚本控制卡片** (80px)
   - 运行/停止按钮（居中）

3. **脚本选项卡片** (60px)
   - 最小化选项
   - 文档/补全选项
   - 错误提示

4. **转换工具卡片** (自适应)
   - 容差设置
   - 导出选项

## 设计原则

1. **简洁优先**：去除不必要的装饰，专注功能
2. **一致性**：统一的配色、间距、圆角
3. **层次感**：通过卡片和间距区分功能区域
4. **易用性**：更大的按钮，清晰的标签
5. **专业感**：现代化的浅色主题
